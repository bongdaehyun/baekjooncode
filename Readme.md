# 🔫 알고리즘을 위해 공부한 사이트 및 코드 들 모음

- 백준 : 브론즈~골드 사이 문제들을 풀었던 소스코드
- 프로그래머스 : 1~3레벨 까지의 문제들의 소스코드
- ICT : 학점연계형 실습을 위해 테스트를 보았던 문제들의 소스코드
- LINE 2021 : 상한기에 풀었던 코드
- swea : 삼성 기출 문제 및 d1~d6 까지의 레벨의 문제들의 소스코드

3. 분할 정복
4. 이진 검색
5. 백트레킹

7. 문자열 패턴 알고리즘 - KMP
8. 동적 프로그래밍 
9. Stack & queue
10.  시물레이션
